# Payment Terminal Simulator

Simulador simples de pagamento em maquininha de cartão desenvolvido em Python.

O programa simula operações de pagamento com cartão, permitindo calcular parcelamentos e possíveis juros.

## Funcionalidades

- simulação de pagamento com cartão
- cálculo de parcelamento
- aplicação de juros dependendo do número de parcelas
- cálculo do valor final da compra

## Tecnologias utilizadas

- Python

## Objetivo do projeto

Projeto desenvolvido para praticar lógica de programação, estruturas condicionais e cálculos financeiros em Python.
